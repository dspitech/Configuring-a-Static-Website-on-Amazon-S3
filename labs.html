Catalogue des laboratoires

Explorez l’ensemble des labs proposés dans le cadre du cursus Cloud Computing à l’ESTIAM.

Labs fondamentaux

Les briques de base pour découvrir AWS et se familiariser avec la console.

Prise en main d’IAM

Création d’utilisateurs, de groupes et de rôles IAM pour gérer les droits d’accès de manière sécurisée.

  • Niveau : Débutant
  • Durée estimée : 45 min

EC2 & Sécurité

Lancer une instance Linux, configurer les Security Groups et se connecter en SSH.

  • Niveau : Débutant
  • Durée estimée : 1 h

Introduction à S3

Créer un bucket, gérer les objets, les politiques d’accès et comprendre les classes de stockage.

  • Niveau : Débutant
  • Durée estimée : 45 min

Labs orientés projets

Construisez des architectures complètes en combinant plusieurs services AWS.

Hébergement d’un site statique avec S3 & CloudFront

Mettre en ligne un site web statique sur S3, le distribuer via CloudFront et ajouter un domaine personnalisé.

  • Niveau : Intermédiaire
  • Durée estimée : 1 h 30

Application avec RDS

Déployer une base de données MySQL managée et y connecter une application web simple.

  • Niveau : Intermédiaire
  • Durée estimée : 2 h

Architecture haute disponibilité

Mettre en place une architecture multi-AZ avec Auto Scaling et Load Balancing.

  • Niveau : Avancé
  • Durée estimée : 2 h 30

Organisation des labs

Les laboratoires sont conçus pour suivre la progression du module de cours. Chaque lab indique clairement ses prérequis, les services AWS mobilisés et les risques éventuels liés à la facturation.

Bonnes pratiques

  • Relire les prérequis avant de démarrer.
  • Supprimer les ressources à la fin du lab pour éviter les coûts inutiles.
  • Travailler avec un compte dédié aux labs lorsque c’est possible.
