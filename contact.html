Problème sur un lab AWS

Notez la région AWS utilisée, le service concerné et, si possible, le message d’erreur exact. Joindre une capture d’écran accélère fortement l’analyse.

Question pédagogique

Précisez le nom du cours, le lab concerné et le contexte (séance, travail individuel, projet) afin d’orienter votre demande vers le bon interlocuteur.

Problème d’accès ou de compte

Indiquez votre promo, votre adresse e‑mail institutionnelle et, le cas échéant, l’identifiant du compte AWS qui pose problème.
